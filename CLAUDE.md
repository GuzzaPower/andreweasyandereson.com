# andreweasyandereson.com

Personal brand and authority site for Andrew "Easy" Anderson.

## Stack

- Static **HTML / CSS / JS only**.
- **No frameworks, no build tools, no CMS.**
- Shared stylesheet at `assets/css/style.css` — linked by every page.

## Design rules (non-negotiable)

- **Color scheme: greens and blues.** No other accent hues.
- **Never use italics anywhere** (no `<em>`/`<i>` italic styling, no `font-style: italic`).
- **Minimum font size 16px everywhere** — no text smaller than 16px.
- **All pages must be WCAG 2.2 AA compliant.**
- **Font colors must pass contrast requirements** (4.5:1 normal text, 3:1 large text / UI).

## Site structure

```
index.html
contact.html
work.html
blog/index.html
blog/[slug].html
privacy-policy.html
terms.html
cookie-policy.html
accessibility.html
assets/css/style.css
```
